# Zoo OOP (Java)

Proyecto simple de zoológico desarrollado en Java para demostrar conceptos de programación orientada a objetos.

## Conceptos utilizados

- Clases abstractas
- Herencia
- Polimorfismo
- Encapsulamiento

## Estructura del proyecto

Animal → clase abstracta base  
Mamifero → subclase  
Ave → subclase  
Insecto → subclase  

MainZoologico → clase principal que ejecuta el programa

## Tecnologías

- Java
- Programación Orientada a Objetos (OOP)
